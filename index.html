How to play:

Each game contains a three-letter string that you must construct words with. The game consists of eight stages, where you must adhere to the syllable count provided. Some stages allow you to add letters to the front of the string, and others to the end. Some allow both.

syllables required: 3

Most stages will have multiple possible answers, so don't be afraid to guess! Each stage completed is worth ONE point.

You'll receive additional points for each OBSCURE word you find. Obscurity is defined as having a Zipf value of 3 or less - which corresponds to words with frequencies of one per million words. As such, you will receive THREE additional points for each obscure word.

syllables required: 3

Obscure word!

Finally, there are GOLDEN words - ONE per stage. These are specifically selected by the game's creator, and award FIVE additional points.

syllables required: 3

Try to complete all eight stages and score as high as possible!
